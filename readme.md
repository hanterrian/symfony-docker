# Symfony simple docker file

```shell
ter - is alias that looks like sail in laravel

ter build (--no-cache)   - Build docker image
ter up -d                - Up docker image
ter composer             - composer alias
ter (bin/)console        - symfony console alias
ter npm                  - npm console alias
ter shell|bash)          - current user bash
ter root-shell|root-bash - root user bash
ter csfix                - PHP-CS Fixer
ter php                  - php command execute
ter bin                  - ./vendor/bin/
```
